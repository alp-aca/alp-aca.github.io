Today we have released ALP-aca v1.0:

To install from PyPI, use the command

```bash
pip install alpaca-ALPs
```

The manual has been uploaded to arXiv, [2508.08354 [hep-ph]](https://arxiv.org/abs/2508.08354). Additionally, a reference guide can be found [here](https://alpaca-alps.readthedocs.io/latest/).
